基于 **盛卡恩门禁统计** WEB页面进行人员流动监控通知。基于Python 3.8 版本编写。

### 主要文件
> * main.py     脚本源代码
> * tmp/		运行目录
> * mark.txt    关注人员ID
> * record		默认快照目录
> * config.ini  配置文件
> * requirements.txt    Python依赖包
> * start.bat    Windows下自动运行脚本
